# Python Café Billing System

A Python CLI-based café billing system that simulates a real-world ordering and billing workflow. The application allows users to place orders using item codes, automatically calculates bills, applies tips and discounts, supports bill splitting, and generates a formatted invoice.

---

## Features

* Menu-driven ordering using item codes
* Dynamic quantity selection
* Automatic subtotal calculation
* Tip calculation based on user input
* Discount calculation using predefined billing rules
* Bill splitting for multiple customers
* Formatted invoice generation in the terminal

---

## Technologies Used

* Python 3
* Command Line Interface (CLI)
